# Card
卡片组件

### 示例
<br/>
<a-card imgSrc="/vuepress.png" summary="vuepress"></a-card>

### 代码

```html
<a-card imgSrc="/vuepress.png" summary="vuepress"></a-card>
```

### 属性
| 参数 | 说明 | 类型 | 是否必填 | 默认值 |  
| :--: | :--: | :--: | :--: | :--: |
| width | 卡片宽度 | Number | false |  -  |
| imgSrc | 图片资源 | String | true |  -  |
| imgHeight | 图片高度 | Number | false |  -  |
| summary | 卡片摘要 | String/Slot | false |  -  |
| footer | 卡片底部 | Slot | false |  -  |
